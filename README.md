#Проект "Анализатор страниц"
**Описание:**

Проект представляет собой web-приложение для анализа сайтов на SEO-пригодность.

***Сборка:***
```java
make build
```
***Тест:***
```java
make test
```
***Sonar:***
```java
make sonar
```
***CheckStyle:***
```java
make lint
```
***Запуск:***
```java
make run
```
***Приложение будет доступно по адресу:*** http://localhost:7070
